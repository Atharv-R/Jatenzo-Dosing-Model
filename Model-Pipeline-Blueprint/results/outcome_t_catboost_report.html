Jatenzo dose model - metrics (catboost, target=outcome_T)

Statistical accuracy

R2 (final T) is against actual OUTCOME_T. Raw T variance is dominated by regression-to-the-mean noise, so it can look weak even with real skill. R2 (delta T) scores predicting the change and is the more honest read. RMSE/MAE/MSE/%-within are identical either way (current_T cancels).

Dose-recommendation accuracy (inverse recommendation test)

For each row we know the dose given and the T achieved. Setting desired_T to the achieved T and asking the recommender for a dose: does it match reality? 'Clean subset' restricts to rows whose actual next dose is a real marketed rung.

Dosing behavior (over/under, mis-dose, multi-rung)

% over/under/exact/mis-dosed compare the recommended dose to the dose actually taken (accuracy). % >1 rung from current is a guideline check (titration should avoid multi-rung jumps) and is a behavioral property of the recommender vs the patient's current dose.
